
*箭頭函式
其函式的本體、參數、回傳值。主要的核心概念沒有變化
-------------------------------------------
普通函式eg.1:
function 函式名稱(函式參數列表){
    程式碼;
    return 回傳值;
};
普通函式eg.2:
let add=function(n1,n2){
    let result = n1 + n2;
    return result;
};

!箭頭函式基本寫法:
let add=(n1,n2)=>{
    let result = n1 + n2;
    return result;
};
-----------------------------------------
!箭頭函式-回傳值簡化語法(只有回傳值敘述，沒有其他指令時):
let 函式名稱=(函是參數列表)=>(回傳值);

eg.
let add=(n1,n2)=>{
    return n1+n2;
};
以上可簡化為:
let add=(n1,n2)=>(n1+n2);
